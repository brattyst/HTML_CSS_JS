## 📌 학습 목표

* HTML로 웹 페이지 구조 이해 및 작성 능력 습득
* CSS로 웹 페이지 스타일링 및 레이아웃 구성 능력 습득
* JavaScript로 동적인 웹 페이지 구현 능력 습득
* JavaScript의 심화 개념 (객체, 함수, 모듈) 이해

## 🚀 개인 목표

* 매일 최소 1시간 이상 학습
* 각 주차별 주요 개념 정리 및 실습
* 주차별 미션 구현 및 복습

## 📌 학습 계획
